# 카드 프리팹 분리 가이드

## 개요
손패용과 필드용 카드 프리팹을 분리하여 각각의 RectTransform 설정을 독립적으로 관리합니다.

---

## 1단계: 필드용 카드 프리팹 생성

### 1-1. 기존 카드 프리팹 복사
1. Project 창에서 기존 `Card.prefab` 선택
2. **Ctrl+D** (또는 우클릭 → Duplicate)
3. 이름을 `Card_Field.prefab`으로 변경

### 1-2. 필드용 프리팹 RectTransform 설정
1. `Card_Field.prefab` 더블 클릭하여 프리팹 편집 모드 진입
2. 최상위 GameObject (Card) 선택
3. Inspector에서 **RectTransform** 설정:

**권장 설정:**
- **Anchor Presets**: Middle Center (Alt+Shift+클릭)
- **Width**: 190 (슬롯 크기에 맞춤)
- **Height**: 190 (슬롯 크기에 맞춤)
- **Pivot**: (0.5, 0.5) - 중앙
- **Anchored Position**: (0, 0) - 초기 위치는 0

**중요:**
- 손패용 프리팹과 크기가 다를 수 있음
- 슬롯 크기와 정확히 일치시켜야 함

### 1-3. 손패용 프리팹 확인
1. 기존 `Card.prefab` 확인
2. 손패에 최적화된 크기 유지 (예: 200x300)
3. 손패 레이아웃에 맞는 설정 유지

---

## 2단계: 코드 수정

### 2-1. DeckManager 수정
필드용 프리팹 필드 추가

### 2-2. FieldSlotManager 수정
필드에 카드 배치 시 필드용 프리팹 사용

### 2-3. GameManager 수정
적 카드 소환 시 필드용 프리팹 사용

---

## 3단계: 프리팹 차이점

### 손패용 프리팹 (Card.prefab)
- **크기**: 손패에 맞춘 크기 (예: 200x300)
- **용도**: 손패에서 드로우된 카드
- **위치**: 손패 영역에 배치

### 필드용 프리팹 (Card_Field.prefab)
- **크기**: 슬롯 크기와 일치 (예: 190x190)
- **용도**: 필드에 소환된 카드
- **위치**: 슬롯에 정확히 배치

---

## 4단계: 테스트

1. 손패에서 카드 드로우 → 손패용 프리팹 사용 확인
2. 필드에 카드 배치 → 필드용 프리팹 사용 확인
3. 카드 위치가 슬롯과 정확히 일치하는지 확인

---

## 주의사항

1. **두 프리팹 모두 같은 컴포넌트 필요**
   - CardDisplay
   - Draggable
   - 기타 필요한 컴포넌트

2. **비주얼 요소는 동일하게 유지**
   - handVisual, boardVisual 등은 동일하게 설정
   - 데이터는 동일하게 표시

3. **크기만 다르게 설정**
   - RectTransform의 sizeDelta만 다르게
   - 나머지는 동일하게 유지
