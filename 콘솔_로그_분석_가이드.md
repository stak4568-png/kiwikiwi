# 콘솔 로그 분석 가이드

## 일반적인 로그 패턴

### 1. 필드 비주얼 관련 경고
```
[CardDisplay] boardVisual이 설정되어 있지만 boardVisualPrefab이 없습니다.
[CardDisplay] 필드 비주얼이 없습니다!
[CardDisplay] 필드 비주얼이 null입니다!
```
**원인**: 필드 비주얼 프리팹이 할당되지 않음
**해결**: `Card_Field.prefab`의 CardDisplay 컴포넌트에 `Board Visual Prefab` 할당

### 2. 필드용 프리팹 관련 오류
```
[FieldSlotManager] 필드용 카드 프리팹이 설정되지 않았습니다.
[FieldSlotManager] 필드용 카드 프리팹이 없습니다.
```
**원인**: DeckManager의 `cardFieldPrefab`이 할당되지 않음
**해결**: Unity 에디터에서 DeckManager의 `Card Field Prefab` 필드에 `Card_Field.prefab` 할당

### 3. 카드 배치 실패
```
[FieldSlotManager] 카드 배치 실패: ...
[FieldSlotManager] 카드 또는 슬롯에 RectTransform이 없습니다.
```
**원인**: 
- 슬롯이 이미 점유됨
- RectTransform이 없음
- 슬롯 인덱스를 찾을 수 없음

### 4. 스펠 카드 관련
```
[스펠 카드] 마나가 부족합니다.
[스펠 카드] 유효한 타겟이 없습니다.
[스펠 카드] EffectManager.instance가 null입니다!
```
**원인**: 
- 마나 부족
- 타겟이 없음
- EffectManager가 씬에 없음

## 로그 공유 방법

Unity 에디터에서:
1. **Console** 창 열기
2. 오류/경고 메시지 선택
3. **Ctrl+C**로 복사
4. 여기에 붙여넣기

또는:
- 전체 콘솔 로그를 텍스트 파일로 저장하여 공유
