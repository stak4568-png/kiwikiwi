# 스펠 카드 만들기 가이드

## 개요
스펠 카드는 손패에서 클릭하면 즉시 효과가 발동되는 카드입니다. 사용 후 제거됩니다.

---

## 단계별 제작 방법

### 1단계: 효과 에셋 생성

스펠 카드에 사용할 효과를 먼저 만들어야 합니다.

#### 1-1. 데미지 효과 만들기
1. Unity 에디터에서 **Project 창**에서 `Assets/card/effect` 폴더로 이동
2. **우클릭** → **Create** → **TCG** → **Effects** → **Damage**
3. 생성된 에셋 선택 후 Inspector에서 설정:
   - **Effect Name**: 효과 이름 (예: "화염구")
   - **Effect Description**: 효과 설명
   - **Timing**: **Manual** (중요!)
   - **Target Type**: 
     - `SingleEnemy`: 적 하나 선택
     - `AllEnemies`: 모든 적
     - `RandomEnemy`: 랜덤 적 하나
     - `EnemyHero`: 적 영웅
   - **Damage Amount**: 데미지량
   - **Can Target Hero**: 영웅 타겟 가능 여부

#### 1-2. 다른 효과 타입들
- **Heal Effect**: 회복 효과
- **Draw Effect**: 카드 드로우
- **Stat Modify Effect**: 스탯 변경
- **Keyword Effect**: 키워드 부여

---

### 2단계: 스펠 카드 데이터 생성

#### 2-1. CardData 에셋 생성
1. **Project 창**에서 `Assets/card/spell` 폴더로 이동
2. **우클릭** → **Create** → **TcgEngine** → **CardData**
3. 파일 이름 지정 (예: `fireball`)

#### 2-2. 기본 정보 설정
Inspector에서 다음 항목들을 설정:

**Display 섹션:**
- **ID**: 고유 ID (예: "fireball")
- **Title**: 카드 이름 (예: "화염구")
- **Art Full**: 카드 일러스트
- **Art Censored**: 검열된 일러스트 (선택)
- **Art Board**: 필드용 일러스트 (선택)

**Stats 섹션:**
- **Type**: **Spell** (중요!)
- **Mana**: 마나 비용 (예: 1)
- **Attack**: 0 (스펠은 공격력 없음)
- **HP**: 0 (스펠은 체력 없음)
- **Lust Attack**: 0 (보통 스펠은 유혹 공격력 없음)

**Card Text 섹션:**
- **Text**: 카드 설명
- **Text Censored**: 검열된 설명 (선택)

#### 2-3. 효과 연결
**Abilities & Effects 섹션:**
1. **Effects** 리스트의 **+ 버튼** 클릭
2. 1단계에서 만든 효과 에셋을 드래그 앤 드롭
3. 여러 효과를 추가할 수 있음

**중요:** 효과의 **Timing**이 **Manual**로 설정되어 있어야 합니다!

---

## 예시: 화염구 스펠 카드

### 효과 설정 (DamageEffect)
```
Effect Name: 화염구
Effect Description: 적 하나에게 2 데미지
Timing: Manual
Target Type: SingleEnemy
Damage Amount: 2
Can Target Hero: true
```

### 카드 설정 (CardData)
```
ID: fireball
Title: 화염구
Type: Spell
Mana: 1
Attack: 0
HP: 0
Text: 적 하나에게 2 데미지
Effects: [위에서 만든 DamageEffect]
```

---

## 사용 방법

1. 게임 중 손패에 스펠 카드가 있으면
2. 카드를 **클릭**하면 효과가 발동됩니다
3. 타겟이 필요한 경우 (SingleEnemy 등):
   - 가능한 타겟이 **하이라이트**됩니다
   - 타겟을 **클릭**하면 효과가 적용됩니다
   - **ESC 키**로 취소 가능
4. 사용 후 카드는 자동으로 제거됩니다

---

## 주의사항

### ✅ 올바른 설정
- **Type = Spell**
- **Timing = Manual**
- **Mana 비용 설정**
- **효과를 Effects 리스트에 추가**

### ❌ 잘못된 설정
- Type이 Spell이 아님
- Timing이 Manual이 아님
- 효과가 Effects 리스트에 없음
- 마나 비용이 0이 아닌데 마나가 부족함

---

## 효과 타이밍 설명

- **Manual**: 손패에서 카드를 클릭할 때 발동 (스펠 카드용)
- **OnSummon**: 필드에 소환될 때 발동 (캐릭터 카드용)
- **OnAttack**: 공격할 때 발동
- **OnDeath**: 죽을 때 발동
- **OnTurnStart**: 턴 시작 시 발동

스펠 카드는 **반드시 Manual**이어야 합니다!

---

## 효과 타겟 타입 설명

- **None**: 타겟 없음 (즉시 발동)
- **SingleEnemy**: 적 하나 선택
- **SingleAlly**: 아군 하나 선택
- **AllEnemies**: 모든 적
- **AllAllies**: 모든 아군
- **RandomEnemy**: 랜덤 적 하나
- **EnemyHero**: 적 영웅
- **PlayerHero**: 플레이어 영웅

---

## 팁

1. **효과 재사용**: 같은 효과를 여러 카드에 사용할 수 있습니다
2. **복합 효과**: 하나의 카드에 여러 효과를 추가할 수 있습니다
3. **마나 밸런스**: 강한 효과는 높은 마나 비용을 설정하세요
4. **타겟 선택**: SingleEnemy/SingleAlly는 타겟 선택이 필요합니다

---

## 문제 해결

### 스펠 카드가 작동하지 않을 때
1. Type이 Spell인지 확인
2. 효과의 Timing이 Manual인지 확인
3. 효과가 Effects 리스트에 있는지 확인
4. 마나가 충분한지 확인

### 타겟 선택이 안 될 때
1. Target Type이 SingleEnemy 또는 SingleAlly인지 확인
2. 적/아군 카드가 필드에 있는지 확인
3. 콘솔 로그 확인
