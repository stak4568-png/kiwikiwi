# 프리팹 분리 코드 수정 사항

## DeckManager.cs 수정

다음 필드를 추가하세요:

```csharp
[Header("카드 프리팹")]
[Tooltip("손패에 드로우되는 카드 프리팹")]
public GameObject cardPrefab; // 손패용
[Tooltip("필드에 소환되는 카드 프리팹 (슬롯 크기에 맞춤)")]
public GameObject cardFieldPrefab; // 필드용
```

## Unity 에디터 설정

1. **DeckManager** 선택
2. Inspector에서 **Card Field Prefab** 필드에 `Card_Field.prefab` 할당
3. **FieldSlotManager** 선택 (선택사항)
4. Inspector에서 **Field Card Prefab** 필드에 `Card_Field.prefab` 할당

## 사용 방법

### 자동 사용 (권장)
- `FieldSlotManager.CreateAndPlaceCard()` 메서드 사용
- 자동으로 필드용 프리팹 선택

### 수동 사용
```csharp
GameObject prefab = DeckManager.instance.cardFieldPrefab;
GameObject card = Instantiate(prefab);
```
