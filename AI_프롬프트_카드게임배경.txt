카드게임 배경 일러스트 - "기억의 거울" 컨셉

[기본 정보]
- 해상도: 2560x1440 (16:9 비율)
- 스타일: 판타지 디지털 페인팅, 신비롭고 몽환적
- 파일 형식: PNG, 고해상도

[레이아웃 구성]

1. 상단 1/3 영역 - 적 영역
   - 색상: 어두운 보라-빨강 톤 (#3d1a2a, #4a1f2f)
   - 우측 상단: 적 초상화 위치 (검은색 박스, 흰색 테두리)
   - 우측 중앙: 적 덱 위치 (흰색 박스, 검은색 테두리)
   - 우측 하단: 적 손패 위치 (흰색 박스, 검은색 테두리)
   - 분위기: 어둡고 위협적인 느낌

2. 중앙 1/3 영역 - 전투 영역 (거울)
   - 핵심 컨셉: "기억의 거울" - 거울이나 수면 같은 반사면
   - 중앙에 거울 균열이나 반사 효과로 좌우 구분
   - 좌측: 플레이어 카드 슬롯 5개 (약한 청록색 글로우)
   - 우측: 적 카드 슬롯 5개 (약한 보라색 글로우)
   - 배경: 반투명 은색/보라색 그라데이션, 거울 느낌
   - 효과: 떠다니는 기억의 파편, 빛나는 파티클
   - 중앙 경계선: 거울 균열이나 수면 파문 효과

3. 하단 1/3 영역 - 플레이어 영역
   - 색상: 어두운 청록-은색 톤 (#1a2d3d, #2a3d4d)
   - 좌측 하단: 플레이어 초상화 위치 (검은색 박스, 흰색 테두리)
   - 좌측 상단: 자원 표시 영역
     * 집중력 표시 (99)
     * 흥분도 표시 (90) - 프로그레스 바 포함
     * 마나 표시
   - 분위기: 차분하고 신비로운 느낌

[시각적 요소]

색상 팔레트:
- 주 색상: 어두운 보라색 (#2d1b3d), 청록색 (#4a9ba8)
- 강조 색상: 은색 하이라이트, 금색 파티클
- 적 영역: #3d1a2a (어두운 보라-빨강)
- 플레이어 영역: #1a2d3d (어두운 청록)
- 중앙 거울: 반투명 은색/보라 그라데이션

"기억의 거울" 연출:
- 중앙 경계선: 거울 균열, 수면 파문, 반사 효과
- 배경: 떠다니는 기억의 파편 (반투명, 빛나는 조각들)
- 파티클: 은색/청록색 빛나는 파티클이 배경에 흩어져 있음
- 분위기: 몽환적이고 신비로운, 약간 어두운 톤

카드 슬롯:
- 좌우 각각 5개씩 슬롯 표시
- 빈 슬롯: 약한 글로우 (청록/보라), 아웃라인만
- 슬롯은 원형 또는 육각형 형태
- 중앙 거울 경계선을 기준으로 좌우 대칭

[기술적 요구사항]

- 상단 20%, 하단 20%는 UI 요소가 올라갈 공간 (텍스트, 버튼 등)
- 중앙 60%가 메인 전투 영역
- 카드가 올라갈 슬롯은 시각적으로 구분되지만 너무 강조하지 않기
- 전체적으로 어둡지만 UI 요소들이 잘 보이도록 적절한 대비
- 거울/반사 효과로 인한 깊이감 표현

[분위기 키워드]
판타지, 신비로운, 몽환적, 거울, 반사, 기억, 파편, 균열, 어두운, 빛나는 파티클, 대칭, 균형, 전투 영역, 카드 게임, TCG

[참고 스타일]
- 어두운 판타지 일러스트
- 거울이나 수면의 반사 효과
- 떠다니는 빛나는 파편
- 신비로운 분위기
- 카드게임 UI 레이아웃
